# RorHash

A bilingual voice application supporting conversational voice chat, smart dictation, and audio file transcription with context awareness.

## Features

- **Voice Chat**: Natural conversation mode supporting Khmer and English
- **Smart Dictation**: Real-time speech-to-text with intelligent formatting
- **Audio Upload**: Transcribe audio files (supports Khmer and English)
- **Real-time Visualization**: Audio waveform visualizer during active sessions

## Tech Stack

- React 19
- TypeScript
- Vite
- Lucide React (icons)

## Prerequisites

- Node.js 18+
- pnpm (or npm/yarn)

## Setup

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd gemini-live_-khmer-&-english
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Create a `.env.local` file in the root directory:
   ```bash
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. Start the development server:
   ```bash
   pnpm dev
   ```


## Usage

### Conversation Mode
Click the microphone button to start a voice conversation. Speak naturally in English or Khmer and receive AI responses.

### Dictation Mode
Switch to dictation mode for speech-to-text transcription with intelligent formatting.

### Upload Mode
Upload audio files for transcription. Supports common audio formats.
